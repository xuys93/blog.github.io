                          Mac常用命令行

                          永生 2019-3-1 3w tools mac

                          # pwd

                          当你键入太多命令，一下找不到自己现在所处于的路径时，pwd可以帮你很好的找到

                          # ys @ xuys in ~/Documents/www/m [12:15:43]
$ pwd
/Users/ys/Documents/www/m
                          1
                          2
                          3

                          # mkdir 新建目录

                          新建目录，可以是以前没有的目录，也可以在已有的目录下追加新的目录，但是值得注意的是，只用mkdir不能同时创建父子目录，如下图中tmp11还没有存在的情况下，不能直接创建,tmp111,如需同时创建多层目录，可以使用mkdir -p

                          # ys @ xuys in ~/Documents/www/m/demo [12:30:17]
$ mkdir tmp

# ys @ xuys in ~/Documents/www/m/demo [12:30:36]
$ mkdir tmp/tmp1

# ys @ xuys in ~/Documents/www/m/demo [12:30:55]
$ mkdir tmp/tmp1/tmp11/tmp111
mkdir: tmp/tmp1/tmp11: No such file or directory

# ys @ xuys in ~/Documents/www/m/demo [12:31:16] C:1
$ mkdir -p tmp/tmp1/tmp11/tmp111

                          # cd 更改目录

                          更改目录，其实也就是指定到对应的目录，这种指定不能跳跃式的，必须一步步的往下指定，如下图不能直接指定到tmp1,要先指定到tmp,在指定到tmp1,也可以直接cd tmp/tmp1 ,可以使用cd直接回到最开始目录，也可以使用cd ..回到上一层目录

                          # ys @ xuys in ~/Documents/www/m/demo [12:31:33]
$ cd tmp

# ys @ xuys in ~/Documents/www/m/demo/tmp [12:32:39]
$ cd tmp1/tmp11/tmp111

# ys @ xuys in ~/Documents/www/m/demo/tmp/tmp1/tmp11/tmp111 [12:32:53]
$ cd ..

# ys @ xuys in ~/Documents/www/m/demo/tmp/tmp1/tmp11 [12:33:13]
$

                          # rmdir 删除路径

                          删除路径,值得注意的是，在删除路径的时候需要指定到它的上一级目录，要不会报错，如下图，指定到了tmp1,可以rmdir下面的tmp11,但是当目录退到tmp时，就不能删除tmp11了，如果遇到你确定是空目录但是它拒绝删除改目录时，可以键入rm -rf<'dir'>即可，其中dir 用实际的目录名替代就可以

                          # ys @ xuys in ~/Documents/www/m/demo/tmp/tmp1/tmp11 [12:33:40]
$ rmdir tmp1
rmdir: tmp1: No such file or directory

# ys @ xuys in ~/Documents/www/m/demo/tmp/tmp1/tmp11 [12:35:02] C:1
$ mkdir tmp1

# ys @ xuys in ~/Documents/www/m/demo/tmp/tmp1/tmp11 [12:35:48]
$ ls
tmp1   tmp111

# ys @ xuys in ~/Documents/www/m/demo/tmp/tmp1/tmp11 [12:35:57] C:127
$ rmdir tmp1

# ys @ xuys in ~/Documents/www/m/demo/tmp/tmp1/tmp11 [12:36:06]

                          # pushd和popd

                          pushd保存当前路径并转到一个新的路径
                          popd回到先前保存的路径

                          # ys @ xuys in ~/Documents/www/m/demo/tmp/tmp1/tmp11 [12:37:36]
$ pushd tmp111
~/Documents/www/m/demo/tmp/tmp1/tmp11/tmp111 ~/Documents/www/m/demo/tmp/tmp1/tmp11 ~/Documents/www/m/demo/tmp ~/Documents/www/m/demo ~/Documents/www/m ~/Documents/www ~/Documents/www/git ~

# ys @ xuys in ~/Documents/www/m/demo/tmp/tmp1/tmp11/tmp111 [12:38:12]
$ popd
~/Documents/www/m/demo/tmp/tmp1/tmp11 ~/Documents/www/m/demo/tmp ~/Documents/www/m/demo ~/Documents/www/m ~/Documents/www ~/Documents/www/git ~
