            # 切换镜像源

            当你把一个项目clone下来想要运行它时，我们都会先npm install一下，但有些项目你会发现一直在下载没有个头头还一直不动，咋怎干等？不。。可以尝试用下它“nrm”。

            nrm是什么鬼呢？
                就是一个切换镜像源工具。
            怎么用？
            当然是先全局安装， npm install nrm -g
            然后捏？
            nrm ls 查看当前镜像源
            再然后捏？
            切换你想要运用的镜像源（如淘宝）nrm use taobao
            就这样easy啦。。。

            三步曲：
npm install nrm -g
nrm ls
nrm use XXX

            # nvm node 包管理

            node

            Nvm (Node Version Manger) 非常好用的管理node版本工具。目前而言，由于node4/6/8的关系，推荐使用nvm管理node版本，便于需要时切换。

            快速安装步骤

            在终端里执行以下命令

            curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.33.8/install.sh | bash
            1

            重启终端。

            nvm install 8 即可安装任意版本的node。推荐装8。
            nvm alias default 8 将上述安装的版本设置为默认版本
            完整文档可以参阅 nvm - github
